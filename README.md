## Customization
Colors - Edit style.css:
css--primary-color: #2e2e2e;
--accent-color: #d4a574;
Stats - Change numbers in index.html:
html<h2 class="stat-number" data-target="250">0</h2>
Products/Testimonials - Just copy and paste existing cards, then update the content.
Libraries Used

Locomotive Scroll (smooth scrolling)
GSAP (animations)
Remix Icons (icons)

All loaded from CDN - nothing to install!

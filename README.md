## Customization
Colors - Edit style.css:<br>
css--primary-color: #2e2e2e;<br>
--accent-color: #d4a574;<br>
Stats - Change numbers in index.html:<br>
html<h2 class="stat-number" data-target="250">0</h2><br>
Products/Testimonials - Just copy and paste existing cards, then update the content.<br>
Libraries Used<br>

Locomotive Scroll (smooth scrolling)<br>
GSAP (animations)<br>
Remix Icons (icons)<br>

All loaded from CDN - nothing to install!
